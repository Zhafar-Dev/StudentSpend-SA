# StudentSpend-SA

## Project Overview
StudentSpend-SA is a simple expense tracking application designed to help users manage their daily spending, categorize expenses, and monitor their budget.

## Purpose
The purpose of this system is to provide a lightweight and user-friendly solution for students to track expenses, even on low-end devices.

## Key Features (Planned)
- Add and manage expenses  
- Categorise spending (food, transport, etc.)  
- Budget tracking and alerts  
- Simple and user-friendly interface  

## Target Users
Students and individuals who need a simple budgeting tool.

## Development Status
Currently in the planning phase. No development has started yet.
